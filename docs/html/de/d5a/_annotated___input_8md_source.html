1 ## Annotated input file description ## {#Annotated}
2 
3 This page describes the allowed input parameters, and a description of what they do. Please note the following:
4 
5 - Note that input-file keywords are case-sensitive - everything should be lower-case!
6 - In the input file, lines beginning with '#' are comments.
7 - Blank lines are ignored.
8 - Keywords can appear in any order.

87 ## Constraints control
88 
89 This block controls the constrained atoms and degrees-of-freedom in GDS calculations and geometry-optimization calculations.
90 
91 The *dofconstraints* input gives the number of fixed DOFs (in this case, 6), with following line in the input file listing the integer-values of the fixed DOFS. Here, counting starts at 1 for the *x*-coordinate of atom 1. The *z*-coordinate of atom 2 has the integer value of 6, and so on...
92 
93  dofconstraints 6
94  1 2 3 5 6 9
95 
96 The *atomconstraints* has the same format as *dofconstraints*, but the integers now refer to the number of fixed atoms, and the indices of the fixed atoms. In the following, 1 atom is fixed - atom number 13.
97 
98  atomconstraints 1
99  13
100 
101 Finally, the *alignedatoms* input allows one to define three atoms which will be used to position and orient any input molecular structure in space. This is very useful when performing NEB calculations as it helps remove overall translations and orientations of the molecules in the optimized string. Note that the NEB routines will automatically select 3 atoms to position and orient if you don't input anything as *alignedatoms*. In the following example, atoms 1, 2 and 3 are chosen to define the relative position and orientation.
102 
103  alignedatoms
104  1 2 3

170 **valencerange** :: Defines the allowed valence ranges of each element as follows:
171 
172  valencerange{
173  C 1 4
174  O 1 2
175  H 0 1
176  Pt 0 3 fz
177  }
178 The fz keyword applies only to frozen atoms, either by being defined in the fixedbonds{} input (see below), or by atomcontraints keyword. In the example,
179 only Pt-X bonds (X= whatever) which can be formed or broken during the GDS calculation are counted into the 'valence'. If a cluster of Pt atoms are not
180 moving during a GDS calculation, and they are bonded to each other (to varying degrees), the above keyword will allow for up to 3 more bonds, not counting the
181 Pt-Pt bonds, to be formed on the Pt atoms.

198 **reactivevalence** :: Defines the valence range one element can have with another
199 
200  reactivevalence{
201  Fe Fe 6 8
202  H C 0 1
203  H O 0 1
204  }
205 
206 In the above, *Fe* must be bonded between 6 to 8 other Fe atoms to be an acceptable molecule. Similarly, the second line indicates that Hydrogen can only
207 be bonded up to 1 carbon atom.
208 
209 **fixedbonds** :: Defines fixed bonds, which are not allowed to change during a chemical reaction (although they can vibrate, translate, etc.)
210 
211  fixedbonds{
212  C O
213  3 4
214  }
215 
216 In the above example, *all* C-O bond orders are fixed at whatever they are in the starting structure. So, if a C-O bond is present in the starting structure, it cannot change during the GDS simulation. Similarly, the bond between atoms *3* and *4* is also fixed at the starting structure value.

236 **molecvalcons** :: Defines the valency that the atoms must have in molecules with all or a subset of the elements defined in each line, with + or minus some possible offset
237 
238  molecvalcons{
239  C 0 ; 0
240  C 4 O 2 ; 2
241  C 4 O 2 H 1 ; 1
242  C 4 N 3 O 2 H 1 ; 2
243  Pt-x C 4 N 3 O 2 H 1 ; 4
244  }
245 
246 If a candidate molecule involved in the reaction in question, might for example contain a Boron atom, none of the contraints apply to such a molecule (none have B in the list)
247 In the above examples; The first line only applies to molecules that are comprised of pure carbon, and setting the valency to 0, with no offset, means that
248 any pure carbon molecules will fail this test.
249 The second line will only apply to atoms with Carbon and/or Oxygens, Carbon can have up to 4 bonds, oxygen 2. Because of the offset 2, carbonate ions might be allowed ( meaning +/- 2 ions or two electron radicals etc... are allowed).
250 The third line would apply to molecules containing C, H and O, so it might allow for the formation of carboxylate anions or its protonated (neutral) counterparts,
251 protonated epoxides, etc... only having one + charge/radical (offset = 1)
252 The fourth line Carbon can have up to 4 bonds, nitrogen 3, oxygen 2 and hydrogen 1. Because of the offset 2 (after the semicolon), the
253 it can allow for the formation of Zwitterions, typical of aminoacids, with a NH4+ and O- terms.
254 The last line applies for scenarios where oragnic molecules are attached to some element with very flexible bon-order/valency - e.g. a platinum cluster can bond flexibly to a varied number of atoms.
255 Consecuently, the suffix "-x" instructs the program not to count the valency of Pt.
256 The variable element (ie Pt) will be discounted ("erased") and each of the submolecules will be judged by the contraints of that line. In other words, if there are two organic moieties attached to the
257 platinum cluster, those moieties will be *individually* screened by the constraints. The bonds to Pt by those other non-Pt atoms attached to Pt will therefore not be counted as bonds,
258 and one should therefore account for this uncertainty by setting a generous offset. In this example, one could envisage a carbon atom could be bonded to three Pt atoms in a cluster,
259 that would still leave an charge offset of 4-3 = 1 on the rest of the organic molecule that carbon is bonded to.
260 
261 In general, for every molecule that is made/broken in a reaction, if a contraint applies to it, then whichever contraint appears first (top to bottom)
262 in the list will determine wheather the molecule is accepted or not (priority is given to the top ones). So placing the first line at the end (in the example)
263 will mean that pure carbon structures would be accepted since line 2 (now being the first) would allow such structures (and the rule would apply to them).
264 All molecules involved in the reaction must be accepted for the reaction to be allowed.
