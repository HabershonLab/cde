3 ## Compiling CDE
4 
5 Before running, CDE must first be compiled.
6 
7 To compile:
8 
9 - Go to the *src* directory.
10 
11 - Edit the *Makefile* so that your Fortran compiler and any required libraries are picked up.
12 
13 - Type *make*.
14 
15 - After compilation, you should now have an executable labelled *cde.x*
16 
17 ## Running CDE
18 
19 To run the code, simply type:
20 
21  cde.x input
22 
23 where *input* is the name of your input file.
24 
25 See the sections @ref Annotated and @ref Example for descriptions of input files.
26 
27 To make things easier, you should add an alias to your *.bashrc* file in your home directory. To do so (assuming a linux/unix system using BASH shell):
28 
29 - Type *cd* to change to your home directory;
30 
31 - Type *vi .bashrc*
32 
33 - Add the line
34 
35 
36  *alias cde='YYY/cde.x'*
37 
38 
39 where *YYY* is the full directory path to the *cde.x* executable.
40 
41 - Save the *.bashrc* file, then type
42 
43 
44  source .bashrc
45 
46 
47 - If successful, you should then be able to type *cde.x* to run the CDE code.
