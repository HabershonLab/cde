Detailed Description

Contains a user-editable routine to calculate the fitness of each molecular structure during a molecular property optimization calculation.

In the subroutine EvaluateFitness(cx), the user must provide a function which provides the fitness value for the chemical structure cx.

IMPORTANT: Larger fitness is better!

IsAcceptable.

Determines whether or not a molecule for optimization is an acceptable chemical structure. To be acceptable, the molecular graph must:

  • Represent a single contiguous molecule,
  • Have no 3-rings,
  • Have no 4-rings.
  • cx: Chemical structure object.
  • flag: Logical flag indicating whether or not structure is acceptable.
